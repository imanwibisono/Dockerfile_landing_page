# landing_page
*Ini adalah landing page sederhana untuk testing*

Ambil file dari github: 
```
git clone https://github.com/imanwibisono/landing_page.git
```
Build Docker Image: 
```
docker build -t landing_page .
```
Jalankan Docker: 
```
docker run -d -p host_port:80 landing_page
```
Selamat mencoba
